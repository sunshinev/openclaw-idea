🏷️ 标签：#AI专题 #GitHub #微软 #语音识别 #开源

📝 详细总结：

微软开源了 VibeVoice 系列语音 AI 模型，包含 ASR（语音识别）和 TTS（语音合成）两大类。VibeVoice-ASR 最大的特点是能一次性处理长达 60 分钟的音频，不像传统 ASR 把音频切成短片段再拼接。模型输出包含说话人识别、时间戳和内容三部分，支持用户自定义关键词来提升特定领域内容的识别准确率。模型原生支持超过 50 种语言，已集成到 Hugging Face Transformers 库中，可以直接调用。此外还提供了 vLLM 推理支持以加速处理。一个叫 Vibing 的语音输入法已经基于 VibeVoice-ASR 开发完成，提供了 macOS 和 Windows 版本。
