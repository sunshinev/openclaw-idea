今日看什么#

今天榜单有点意思。Claude Code 相关项目占了三个席位，看来 Anthropic 的命令行工具热度还在涨。Microsoft 开源的语音 AI 项目 VibeVoice 一天涨了 2400+ stars，这个速度有点吓人。还有就是 Deep-Live-Cam 这种 deepfake 工具常年霸榜，争议一直有，热度一直不掉。

重点项目#

1. microsoft/VibeVoice ⭐ 30,140 (+2,492)#

Microsoft 开源的语音 AI，一天涨两千多 stars。

定位是"Open-Source Frontier Voice AI"，从名字看应该是面向前沿语音处理方向的开源项目。微软开源这个方向值得关注，语音生成、识别这块最近竞争很激烈。

GitHub: https://github.com/microsoft/VibeVoice

2. luongnv89/claude-howto ⭐ 9,805 (+4,232)#

增长最快的项目，一天涨了四千多 stars。

这是一个 Claude Code 的可视化教程，描述写得很清楚：从基础概念到高级 agents，附带可以直接复制使用的模板。这种文档类项目能火，说明大家确实需要学习怎么用好这些工具。

GitHub: https://github.com/luongnv89/claude-howto

3. NousResearch/hermes-agent ⭐ 18,550 (+1,851)#

“The agent that grows with you”——标语挺有意思。

NousResearch 这家公司在 LLM 领域做过不少东西，包括 Hermes 系列模型。这个 agent 项目看名字应该是和他们之前的模型生态有联系。一天涨一千八百 stars，热度不低。

GitHub: https://github.com/NousResearch/hermes-agent

4. hacksider/Deep-Live-Cam#

实时换脸工具，只要一张图片就能做 deepfake。

这类项目常年出现在 Trending 上，说明需求一直存在。不过 deepfake 的伦理争议也一直跟着，技术好用，怎么用好是另一个问题。

GitHub: https://github.com/hacksider/Deep-Live-Cam

5. OpenBB-finance/OpenBB#

金融数据平台，面向分析师、量化交易和 AI agents。

开源金融工具这块竞争少，OpenBB 做了挺久，用户群稳定。现在 AI agents 需要数据接入，这个方向会有更多需求。

GitHub: https://github.com/OpenBB-finance/OpenBB

6. sherlock-project/sherlock#

用用户名搜索社交媒体账号的工具。

安全研究人员常用的东西，输入一个用户名，能在几十个平台上查有没有对应账号。做社工调查、账号追踪的时候很实用。

GitHub: https://github.com/sherlock-project/sherlock

7. apache/superset#

Apache 的数据可视化平台。

老牌项目了，Apache 基金会维护，企业级数据可视化的选择之一。BI 需求还在涨，Superset 作为开源方案，用户基数稳定。

GitHub: https://github.com/apache/superset

8. fastfetch-cli/fastfetch ⭐ 21,405 (+276)#

neofetch 的替代品，系统信息展示工具。

neofetch 不维护之后，fastfetch 接了这个需求。功能更丰富，性能更好，C 语言写的。Linux 用户桌面展示系统信息的时候会用到。

GitHub: https://github.com/fastfetch-cli/fastfetch

今日观察#

Claude Code 热度还在涨。 今天三个 Claude Code 相关项目上榜，说明开发者社区对这个工具的学习需求很高。教程、最佳实践这种"配套内容"也在跟着火。

语音 AI 开源加速。 Microsoft 开源 VibeVoice 一天涨两千多 stars，语音生成、识别方向的开源竞争会越来越激烈。

Agent 概念持续发酵。 Hermes Agent 这种"陪伴型 agent"概念开始出现，不只是执行任务，而是强调"成长"和"长期使用"。
