搞钱副业#

1. karpathy/autoresearch - AI 自动科研助手#

来源: GitHub | 60,060 stars | Python

Andrej Karpathy 又出手了。这个项目让 AI agent 在单 GPU 上自动跑研究实验，从设计思路到代码实现全包。

说白了，一个人也能做过去需要整个实验室才能做的事。过去搞研究得拉投资、租服务器、组建团队，现在？一个脚本的事。对于想靠技术变现的人，会用这类工具放大自己的能力，比啥都强。你不需要变成全栈工程师，只需要学会怎么指挥 AI 替你干活。

2. gstack - 复制 YC CEO 的配置#

来源: GitHub | 54,764 stars | TypeScript

Garry Tan（Y Combinator CEO）的开发环境配置被开源了。15 个 AI 工具，分别扮演 CEO、设计师、工程经理、发布经理、文档工程师和 QA。

与其说是配置，不如说是把顶级投资人的工作流给拆解了。想搞钱的话，别盯着工具本身，要理解工具背后的思维方式。 这套配置本质上是「一个人就是一支军队」的生产系统。

3. vana/noisia - AI 音乐检测器#

来源: Hacker News | 287 comments

Vana 出了个开源工具，能检测音频是不是 AI 生成的，准确率相当可观。

典型「淘金热里卖铲子」的生意。 所有人都想用 AI 做音乐，但谁来鉴别？版权公司、流媒体平台、原创作者都有需求。提前占坑，等风来。
